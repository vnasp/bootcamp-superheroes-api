# Revista de Superhéroes

Aplicación web interactiva estilo revista que permite buscar superhéroes por su ID (del 1 al 731) consumiendo la SuperHero API. Muestra información detallada del personaje incluyendo nombre, alias, género, editorial y primera aparición, junto con un gráfico de radar (Chart.js) que visualiza sus estadísticas de poder (inteligencia, fuerza, velocidad, durabilidad, poder y combate).

> Nota: Este es un proyecto académico desarrollado durante el bootcamp de Talento Digital para Chile.

## Vista previa

Visita la aplicación desplegada en [GitHub Pages](https://vnasp.github.io/bootcamp-superheroes-api/)

## Funcionalidades

- Búsqueda de superhéroes por ID (rango 1-731)
- Validación de entrada del usuario con alerta visual
- Visualización de datos del superhéroe (imagen, nombre, alias, género, editorial, primera aparición)
- Gráfico de radar con estadísticas de poder del héroe
- Diseño responsive con estética de revista de cómics
- Navegación al inicio haciendo clic en el logo

## Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)
- jQuery (manipulación DOM y AJAX)
- Chart.js (gráfico de radar)
- TailwindCSS (estilos responsive)

# Super Héroes Magazine 🦸🏻‍♂️

Bienvenido al repositorio de Super Héroe Magazine, un sitio web interactivo donde conocerás qué super héroe te corresponde en el multiverso o podrás buscar más info sobre alguno en específicol, gracias a la información proporcionada por la API de Superhero.

## Características 🌟

- **Búsqueda de superhéroes por ID**
- **Visualización de datos del superhéroe**
- **Gráfico de radar para ver las estadísticas**
- **Diseño Responsive**

## Tecnologías Utilizadas 💻

Este proyecto está construido utilizando las siguientes tecnologías y bibliotecas:

- **HTML/CSS**: Para estructura y diseño de la interfaz.
- **JavaScript**: Para la lógica de interacción en el cliente.
- **jQuery**: Utilizado para simplificar la manipulación del DOM y el manejo de eventos.
- **AJAX**: Para el consumo de la API SuperHero.
- **Chart.js**: Para generar gráficos de radar que representan las estadísticas de los superhéroes.
- **TailwindCSS**: Para estilos rápidos y responsivos.
